# Research Analytics Dashboard

An interactive data visualization dashboard built to analyze and explore research project funding data using Python and Streamlit.

This project focuses on transforming raw tabular data into meaningful visual insights that support analytical and decision-making use cases.

---

##  Key Features
- Interactive filters for departments, funding agencies, and project status
- KPI cards showing total funding, project count, and average project value
- Department-wise and agency-wise funding analysis
- Visualization of project duration vs funding
- Similar project identification using KNN-based similarity logic
