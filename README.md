# Research Analytics Dashboard

An interactive data visualization dashboard built using Streamlit to analyze research funding data.

## Features
- Interactive filters and KPIs
- Department-wise and agency-wise funding analysis
- Duration vs funding insights
- Similar project identification using KNN logic

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## How to Run
pip install -r requirements.txt  
streamlit run app.py

## Learning Outcomes
- Data preprocessing and visualization
- Building dashboards using Streamlit
- Understanding similarity-based analysis
